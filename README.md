# WI-INSANE AutoInvite

A small auto‑invite helper for Turtle WoW. It watches whispers (and optionally Guild chat) for your keywords and invites.

Version: 1.3 (Turtle WoW 1.18.0)

## Install
- Copy `WI` into `Interface\\AddOns\\`.
- Make sure `WI.toc`, `WI.lua`, and `Icon/wi-icon.tga` exist.
- `/reload` or restart the game.

## Use
- `/wi` opens settings.
- Type a keyword and press `Enter` to add.
- Shift+Click to select many; `Delete` or `Remove` to delete.
- Toggle `Enable auto‑invite` and `Scan guild chat`.
- Minimap button: left‑click opens, right‑click toggles, drag to move.
- Press `ESC` to close.
